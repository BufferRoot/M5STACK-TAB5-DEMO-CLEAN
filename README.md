M5Tab5 User Demo
User demo source code of M5Tab5.
ESP-IDF v5.5.3

A production-ready M5Stack Tap5 framework re-engineered for professional ESP32-S3 development. This version upgrades the ESP-IDF and fixes VS Code "fit issues" like broken header paths. Just download, unzip, and add to VS Code—everything just works. A perfect, low-latency base for any Tap5 project with no additional configuration needed.
